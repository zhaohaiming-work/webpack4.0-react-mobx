# webpack4.0-react-start-kit

* 需要环境
 ```bash
 node > 8.9.0
 yarn >= 1.6.0 
 ```

* 项目配置

> webpack4.0 + react + react-router-dom + mobx

* 项目启动步骤：

```bash

  如果安装了yarn

  1. yarn install ----  安装node包 

  2. yarn start   ----  启动项目

  如果安装了npm
  1. npm install ----  安装node包 

  2. npm start   ----  启动项目

 ```
 * 项目打包：
 
 ```bash

  如果安装了yarn

   yarn build  

  如果安装了npm
  
   npm build

 ```
